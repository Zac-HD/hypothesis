RELEASE_TYPE: minor

This release adds native support for `Tyche
<https://marketplace.visualstudio.com/items?itemName=HarrisonGoldstein.tyche>`__
(`source <https://github.com/tyche-pbt>`__, `paper <https://harrisongoldste.in/papers/uist23.pdf>`__),
a VSCode extension which visualizes the distribution of generated inputs to
your property-based tests and their resulting code coverage.
